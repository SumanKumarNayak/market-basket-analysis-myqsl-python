# Market Basket Analysis with MySQL and Python

## Objective
To discover product association rules from retail transaction data using MySQL and Python.

## Technologies Used
- MySQL
- Python
- Pandas
- mlxtend

## Dataset
Online Retail Dataset (UCI Machine Learning Repository)

## Methodology
- Data cleaned and prepared
- Basket creation using MySQL
- Frequent itemset mining using Apriori algorithm
- Association rule generation based on lift and support

## Results
Top 5 association rules were identified showing strong co-purchase patterns.
